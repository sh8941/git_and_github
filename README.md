# 🚀 Learn Git & GitHub

This repository is created for **learning and practicing Git and GitHub** from scratch.  
It contains examples, notes, commands, and experiments to understand version control concepts practically.

---

## 📚 What This Repo Is For

- Learning **Git basics**
- Understanding **GitHub workflow**
- Practicing:
  - version control
  - branching
  - merging
  - commits
  - pull requests
  - collaboration
- Making mistakes safely and learning from them 😄

---

## 🧠 Topics Covered

### 🔹 Git Basics
- `git init`
- `git status`
- `git add`
- `git commit`
- `git log`
- `git diff`

### 🔹 Branching
- `git branch`
- `git checkout`
- `git switch`
- `git merge`
- merge conflicts

### 🔹 GitHub
- Creating repositories
- Pushing code
- Pulling changes
- Forking
- Pull Requests (PR)
- Issues
- Collaboration flow

### 🔹 Advanced Concepts (Learning Phase)
- Rebase
- Cherry-pick
- Stash
- Reset vs Revert
- Detached HEAD
- Git internals (objects, blobs, trees, commits)
